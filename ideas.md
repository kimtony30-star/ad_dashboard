## 선택된 디자인 철학: "데이터 인텔리전스 다크 테마"

<response>
<text>
**Design Movement:** Data Intelligence / Corporate Dark UI (Bloomberg Terminal meets modern SaaS)

**Core Principles:**
1. 데이터가 주인공 — 차트와 수치가 최대한 넓은 공간을 차지하도록 레이아웃 설계
2. 정보 계층 — 섹션 헤더, KPI 카드, 차트 순으로 시선 흐름을 설계
3. 절제된 색상 — 다크 배경에 청록(teal)/앰버(amber) 포인트 컬러로 강조
4. 밀도 있는 레이아웃 — 사이드바 없이 풀 와이드 그리드 구조

**Color Philosophy:**
- Background: 딥 네이비 (#0f1117) — 집중력 유지, 눈의 피로 감소
- Card: 약간 밝은 네이비 (#1a1f2e)
- Primary Accent: 청록(Teal) — #2dd4bf — 매출 지표 강조
- Secondary Accent: 앰버(Amber) — #f59e0b — 노출 지표 강조
- OCB: #3b82f6 (블루), Syrup: #a855f7 (퍼플), Olock: #10b981 (에메랄드)

**Layout Paradigm:**
- 상단 KPI 요약 카드 3개 (전체 매출, 전체 노출, 기간)
- 탭 기반 섹션 전환 (섹션1: 앱별 / 섹션2: ADPF·네트워크별 / 섹션3: 뉴스)
- 각 섹션: 좌측 라인차트(추이) + 우측 바차트(합계 순위) 2컬럼 레이아웃

**Signature Elements:**
1. 수치 위에 얇은 구분선 + 레이블 — 데이터 밀도감 표현
2. 차트 호버 시 반투명 툴팁 카드

**Typography System:**
- 제목: Pretendard 700 (한글 최적화)
- 수치: JetBrains Mono (숫자 가독성)
- 본문: Pretendard 400
